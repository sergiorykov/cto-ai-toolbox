# research/

Working area for researches — **a workbench, not the final shelf**. Each research lives in a sonorous subfolder; its goal is to enrich the knowledge base. On completion the results always move into a fitting top-level knowledge folder (or a new one is created) — see the lifecycle in [CLAUDE.md](../CLAUDE.md).

Every research package: `xxxx-overview.md` (accessible meaning + proposed further use cases per the [vision](../principles/vision-ai-assisted-cto.md)) + `xxxx-<specific>.md` deep dives + `xxx-case-yyy.md` usage cases. Base docs EN; `-ru` docs enrich with Russian sources.
